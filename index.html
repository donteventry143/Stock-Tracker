About Stock Tracker

Track stocks in real-time, view trending market data, and stay informed with the latest updates. This project showcases stock tracking functionality using APIs and allows users to search for stock symbols and view their performance.

Future Enhancements:

  • Implement a user sign-in feature for personalized experiences.
  • Allow users to create and manage a list of their desired stocks.
  • Add functionality to calculate the total value of a user's stock portfolio.
